#  Biopython Projects

Beginner-friendly bioinformatics scripts developed using Python and Biopython as part of MSc Bioinformatics training.

##  Project 1: DNA Sequence Analyzer

# Description
A Biopython-based tool to analyze DNA sequences and extract biologically meaningful information.

#Features
- Accepts DNA sequence as input
- Calculates sequence length
- Computes GC content (%)
- Generates complement sequence
- Generates reverse complement
- Transcribes DNA -> RNA
- Translates DNA -> Protein

# Technologies Used
- Python
- Biopython
- Google Colab
- GitHub

# Files
- 'DNA_SEQUENCE_ANALYZER.ipynb' - Colab notebook
- 'dna_counter.py' - Python script
